# Dark theme for TypeDoc

Theme Docs: https://typedoc.org/guides/themes/

## Build
To make a build:

```
npm run build
```

This puts the theme you want to use in `dist/dark/`. 


## Usage
```
typedoc --theme path/to/dist/dark/
```


Visit https://typedoc.org/ to learn more about TypeDoc.
